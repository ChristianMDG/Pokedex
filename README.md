# Pokédex React + Vite + TailwindCSS

Ce projet est un Pokédex développé avec **React**, **Vite** et **TailwindCSS**, utilisant l'API officielle [PokeAPI](https://pokeapi.co/api/v2/pokemon) pour afficher les informations des Pokémon.

---

## Fonctionnalités principales

- Affichage d'une liste de Pokémon avec leurs images, types et informations principales.
- Détails complets pour chaque Pokémon : faiblesses, évolutions, descriptions, etc.
- Interface moderne et responsive grâce à TailwindCSS.
- Effets interactifs et animations pour une meilleure expérience utilisateur.
- Déploiement simple via **GitHub Pages** ou **Vercel**.

---

## Installation et lancement

1. **Cloner le dépôt :**

```bash
git clone https://github.com/ChristianMDG/Pokedex.git
cd Pokedex
````

2. **Installer les dépendances :**

```bash
npm install
```

3. **Lancer le serveur de développement :**

```bash
npm run dev
```

Le projet sera accessible par défaut sur [http://localhost:5173](http://localhost:5173).

---

## Construction et déploiement

1. **Construire le projet pour la production :**

```bash
npm run build
```

2. **Déployer sur GitHub Pages :**

```bash
npm run deploy
```

> ⚠️ Assurez-vous que la propriété `homepage` dans le `package.json` et la propriété `base` dans `vite.config.js` soient correctement définies, par exemple `/Pokedex/` si vous déployez sur :
> `https://ChristianMDG.github.io/Pokedex/`

---

## Technologies utilisées

* **React** — bibliothèque JavaScript pour construire l'interface utilisateur.
* **Vite** — outil de build rapide pour les projets frontend.
* **TailwindCSS** — framework CSS utilitaire pour un design rapide et responsive.
* **Axios** — pour effectuer des requêtes HTTP vers l’API PokeAPI.
* **PokeAPI** — API gratuite pour récupérer les données des Pokémon.
